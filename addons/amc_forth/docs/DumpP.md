# DUMPP &emsp; (DumpP)
If 'u' is greater than zero and 'addr' is less than the size of the Parameter Stack, display the contents of 'u' consecutive addresses of the Parameter Stack starting at 'addr'.<br/>Example usage: 500 100 DUMPP
* ( addr u -- )
