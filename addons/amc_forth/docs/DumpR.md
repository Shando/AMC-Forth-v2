# DUMPR &emsp; (DumpR)
If 'u' is greater than zero and 'addr' is less than the size of the Return Stack, display the contents of 'u' consecutive addresses of the Return Stack starting at 'addr'.<br/>Example usage: 500 100 DUMPR
* ( addr u -- )
