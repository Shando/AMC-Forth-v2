# BUFFER: &emsp; (BufferColon)
Create a dictionary entry for <name>, associated with n bytes of space.<br/>Executing <name> will return address of the starting byte on the stack.<br/>Example usage: <n> BUFFER: <name>
* ( 'name' n -- )
