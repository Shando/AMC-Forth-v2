# DUMPRAM &emsp; (DumpRam)
If 'u' is greater than zero and 'addr' is less than the size of the RAM, display the contents of 'u' consecutive addresses of the RAM starting at 'addr'. Example usage: 256 64 DUMPRAM
* ( addr u -- )
