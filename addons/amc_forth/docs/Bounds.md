# BOUNDS &emsp; (Bounds)
Given a memory block represented by starting address 'addr' and length 'u' in address units, produce the end address 'addr' + 'u' and the start address in the right order.<br/>Example usage: 256 10 BOUNDS
* ( addr u -- addr+u addr )
